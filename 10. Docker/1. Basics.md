# 1. Docker Basics

## Test Installation
```bash
docker run hello-world
```
Runs a hello-world container to verify Docker is installed correctly.

---

## Check Docker Version
```bash
docker --version
```
Shows the installed Docker version.

```bash
docker info
```
Shows system-wide Docker information (containers, images, storage, etc.).

---

## Images

```bash
docker images
```
List all images downloaded on your machine.

```bash
docker pull nginx
```
Download the nginx image from Docker Hub.

```bash
docker rmi nginx
```
Remove an image from your machine.

---

## Containers

```bash
docker ps
```
List all **running** containers.

```bash
docker ps -a
```
List **all** containers (running + stopped).
