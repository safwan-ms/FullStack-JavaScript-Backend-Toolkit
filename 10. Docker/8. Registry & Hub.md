# 8. Registry & Docker Hub

Docker Hub is the default public registry for storing and sharing images.

---

## Login & Logout
```bash
docker login
```
Log in to Docker Hub (prompts for username and password).

```bash
docker logout
```
Log out from Docker Hub.

---

## Tag an Image
```bash
docker tag my-app username/my-app:v1
```
Tag a local image so it can be pushed to Docker Hub.

| Part | Description |
|------|-------------|
| `my-app` | Local image name |
| `username` | Your Docker Hub username |
| `my-app` | Repository name on Docker Hub |
| `v1` | Tag / version |

---

## Push an Image
```bash
docker push username/my-app:v1
```
Upload your tagged image to Docker Hub.

---

## Pull Your Image Anywhere
```bash
docker pull username/my-app:v1
```
Download your image on any machine (server, CI, teammate's machine).

---

## Useful Hub Commands
```bash
docker search nginx
```
Search Docker Hub for public images directly from the terminal.

```bash
docker pull nginx:1.25
```
Pull a specific version of an image using a tag (avoid using `latest` in production).
